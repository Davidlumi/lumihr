# lumi.benchmark — Sub-processor List

**Version 1.0 — DRAFT, pending legal review**

This page lists the sub-processors lumi engages to deliver the service. It is
a draft placeholder maintained as an index; the final, reviewed list will name
each sub-processor, the processing purpose, and the location before launch.

## How to read this list

A sub-processor is a third party that processes data on lumi's behalf (for
example, hosting or email delivery). lumi remains accountable for any
sub-processor's handling of data under the Data Sharing Agreement (DPA).
